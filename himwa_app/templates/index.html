Available APIs

1. Index

GET /

This API returns the default documentation page you are currently viewing.

2. Sign In

POST /signin

Allows a user to sign in using email and password. On success, it returns the user details and a token.

Response fields:

  • fullname
  • email
  • phonenumber
  • area_of_residence
  • profile_image_url
  • date_joined
  • token

Example Response:

{ "fullname": "John Doe",
"email": "john@example.com",
"phonenumber": "0712345678",
"area_of_residence": "Nairobi",
"profile_image_url": "https://res.cloudinary.com/...",
"date_joined": "2025-08-20T10:00:00Z",
"token": "eyJhbGciOi..."
}

3. Sign Up

POST /signup

Registers a new member. The user must provide required fields.

Required fields:

  • fullname
  • national_id
  • phonenumber
  • email
  • area_of_residence
  • password
{ "message": "Successfully registered" }

4. Reset Password

POST /resetpassword/<email>

Sends a password reset email to the given address.

{ "message": "A email to reset password is successfully sent" }

5. Get All Members

GET /members

Retrieves a list of all registered members. Requires a valid Firebase token.

Billing APIs

6. Create Bill

POST /create_bill

Create a new bill for a member.

Required fields: member_id, biller_id, units, due_date

{ "message": "Bill created successfully",
"bill_id": 10,
"member": "Jane Doe",
"amount": 200,
"status": "UNPAID",
"due_date": "2025-09-01T00:00:00Z"
}

7. Get All Bills

GET /get_all_bills

Returns a list of all bills.

8. Get Member Bills

GET /get_member_bills/<member_id>

Returns all bills for a given member.

[ {
"id": 5,

"date": "2025-08-15T10:30:00Z",
"due_date": "2025-09-01T00:00:00Z",
"status": "UNPAID"
}
]

9. Pay Bill

GET /pay_bill/<bill_id>/<payment_amount>/<payment_method>/<member_id>

Processes a payment for a specific bill and updates its status.
